# Quick Start

## Prerequisite

- [docker](https://www.docker.com/products/docker-desktop)

## Start Moodle and supporting services

```
docker compose up -d
```

- moodle - http://localhost:8080
- mailcatcher - http://localhost:1080
- postgresql - localhost:5432
